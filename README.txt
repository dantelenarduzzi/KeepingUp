TITLE: Keeping Up

DESCRIPTION:
Keeping Up is a software I've developed that helps consumers 
maintain and keep up with the latest trends regarding gaming.
Whether it be looking for a new game or seeing what everyone 
is play or even looking for a new game that is coming out. 
Keeping up will help you manage all your gaming needs!

GETTING STARTED:
In order to get start all you will need to do is download the app and click on the hamburger menu in the top left to 
navigate through all of the different menu items, You can goto Release dates if youre looking for new games coming in the 
future or if you're looking to see the top games being watched right now theres a tab for that aswell. Theres even a section
 where you can vote for your favorite game and get it featured for the week!

PREREQUISITES:
There are no prerequisites to KeepingUp, just download browse and enjoy!
